
# CardSpin

CardSpin je jednoduchá webová hra, kde môžeš hrať sloty a blackjack.  
Vytvoril som ju ako ročníkový projekt, aby som sa naučil základy webu a logiku hier.

## Čo hra vie
- **Sloty** – náhodné symboly a výhry  
- **Blackjack** – klasické pravidlá, možnosť "Hit" alebo "Stand"  
- **Bodovanie** – sledovanie kreditov hráča  
- **Interaktivita** – tlačidlá a vizuálne zmeny pri akciách
